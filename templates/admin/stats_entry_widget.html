ℹ️ What do these labels mean?
Stats Needed
No player stats have been entered for one or more games yet.
Outcome Needed / Outcome ⚠
Player stats are entered, but a game's result wasn't saved in the Game Outcome section — so the standings don't include that game yet. Open the game and save the Game Outcome.
Goalie Needed / Goalie ⚠
Goals were scored but no goalie goals-against is recorded for one team, so that goalie's GAA / save stats won't count. Open the game and add a stat row for that team's goalie.
Outcome ✓ / Done
Everything for the game is recorded.
Playoff
A postseason game. Results don't affect regular-season standings, so there's no Game Outcome to record — just enter player and goalie stats.
